# Task Manager (Console)

Un gestor de tareas por consola escrito en python, enfocado en practicar POO, CRUD y persistencia en JSON

## Funcionalidades 
- Crear
- Listar (todas / pedientes / completadas)
- actualizar
- borrar
- Persistencia en JSON

## Tecnologias
- Python3
- Programacion orientada a objetos
- manejo de archivos (JSON)
- Git y Github

## Estructura del Proyecto
- tarea.py define la entidad Tarea
- gestor.py logica de negocio (CRUD)
- main.py interfaz por consola y control del flujo

## Estado del proyecto
- En desarrollo

## Objetivo del proyecto
Este proyecto fue creado con fines de aprendizaje, enfocado en:
   - Programacion orientada a objetos (POO)
   - Operaciones CRUD 
   - Persistencia de datos con JSON
